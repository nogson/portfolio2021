## オブジェクト関連

TypeScriptでStringをkeyにしてオブジェクトにアクセスする場合。

```typescript
type User = { [key: string]: string }
const user: User = { name: "sato" };
```

keyの指定をする場合は、key用の型をMapped Typesで割り当てます。  
そのまま割り当てると初期値の設定する必要がある為`?`でOptional設定にします。

```typescript
type Key = "name" | "age";
type User = { [key in Key]?: string }
const user: User = { name: "sato" };
```
