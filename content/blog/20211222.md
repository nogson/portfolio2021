# windowオブジェクトをmock化する

windowサイズを使っている関数などをテストする為に、windowオブジェクトをモック化する。

```
const originalWindow = { ...window }
const windowSpy = jest.spyOn(global, 'window', 'get')
windowSpy.mockImplementation(() => ({
  ...originalWindow,
  innerWidth: 1920,
 innerHeight: 1200
}))

... テストを書く

windowSpy.mockRestore()
```
